# AWS EC2 Nginx Web Hosting

## Overview

This project demonstrates the deployment of a static website on an Amazon EC2 instance using the Nginx web server. The website was created using HTML and hosted on a cloud-based virtual machine.

## Technologies Used

* Amazon EC2
* Nginx
* Linux
* HTML
* Git & GitHub

## Steps Performed

1. Launched an EC2 instance.
2. Connected to the server using SSH.
3. Installed and configured Nginx.
4. Created a webpage using the Nano editor.
5. Hosted the website on the EC2 instance.
6. Verified access through the public IP address.

* Linux Commands
* SSH Connectivity
* Nginx Configuration
* Cloud Computing Fundamentals
